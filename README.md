# NoaTechSolutions Website

Official website project for NoaTechSolutions.

This repository contains the frontend codebase for a modern marketing and web presence focused on digital services, brand positioning, and conversion-oriented user experience.

## Overview

The project is being developed as a polished, portfolio-grade website with emphasis on:

- clear service communication
- responsive navigation and mobile usability
- modern visual hierarchy
- clean component structure
- SEO-ready foundations

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- ESLint

## Current Scope

The codebase currently includes:

- homepage structure and section layout
- responsive resizable navigation
- mobile menu flow with submenu handling
- brand assets integration
- sitemap and robots configuration

## Project Structure

```text
app/
  components/
    ui/
  dashboard/
  globals.css
  layout.tsx
  page.tsx
  robots.ts
  sitemap.ts
public/
docs/
```

## Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

The project is configured to run locally on:

```text
http://localhost:3006
```

Run linting:

```bash
npm run lint
```

Build for production:

```bash
npm run build
```

## Notes

- This repository is intended to evolve as the main website for NoaTechSolutions.
- Visual direction, navigation behavior, and service architecture are under active refinement.
- Brand assets in `public/` are used for interface exploration and production integration.
